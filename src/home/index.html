SQL Injection

Une vulnérabilité critique permettant d'exécuter des commandes SQL malveillantes dans l'application.

Cross-Site Scripting (XSS)

Une vulnérabilité qui permet à un attaquant d'injecter des scripts malveillants dans les pages vues par d'autres utilisateurs.

Cross-Site Request Forgery (CSRF)

Une vulnérabilité permettant à un attaquant de tromper un utilisateur authentifié pour exécuter des actions non autorisées.
